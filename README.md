# 🛡️ Call Blocker - تطبيق حجب المكالمات

## الفكرة
التطبيق بيخلي موبايلك يبان "مغلق" للناس العاديين، بس الأرقام اللي تضيفها في قائمة الاستثناء هتعدي عادي.

## المميزات
- ✅ يرفض كل المكالمات الواردة تلقائياً
- ✅ قائمة أرقام استثناء (Whitelist)
- ✅ يشتغل في الخلفية (Foreground Service)
- ✅ يشتغل تلقائياً بعد إعادة تشغيل الموبايل
- ✅ مفتاح تفعيل/إيقاف سريع
- ✅ بيدي الناس إشارة "مشغول" أو "مغلق" حسب الجهاز

---

## طريقة البناء (Android Studio)

1. **افتح Android Studio**
2. **Open an Existing Project** → اختار فولدر `CallBlocker`
3. انتظر Gradle يخلص
4. **Run** على موبايلك

---

## أول ما يشتغل

1. اقبل كل الصلاحيات المطلوبة
2. ضيف الأرقام اللي تحب تعدّيها (أرقام استثناء)
3. Switch المفعّل يكون ON
4. اقفل الشاشة - التطبيق بيشتغل في الخلفية

---

## الصلاحيات المطلوبة

| الصلاحية | السبب |
|----------|-------|
| READ_PHONE_STATE | عشان يعرف في اتصال واردة |
| ANSWER_PHONE_CALLS | عشان يرفض الاتصال |
| CALL_PHONE | للـ API القديمة |
| FOREGROUND_SERVICE | عشان يشتغل في الخلفية |
| RECEIVE_BOOT_COMPLETED | يشتغل تلقائياً بعد الريستارت |

---

## ملاحظة مهمة
على بعض الأجهزة (Xiaomi, Huawei, Samsung) لازم تروح إعدادات التطبيق وتفعّل:
- **"Allow background activity"**
- **"Autostart"**

عشان الأندرويد ميوقفش التطبيق.
